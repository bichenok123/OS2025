Работа №2
Получение индефекатора пользователем и группы
Цель работы: изучить системные вызовы, возвращающие идентификаторы текущего пользователя и его группы

Для успешной работы необходимых системных вызово на экран информации нужно поделючть библиотеки языка C:
studio.h
sys/types.h
unistd.h
stdlib.h

Для получения идентификаторов пользователя и группы необходимо использовать системные вызовы:
int getuid();
int getgid();

В программе pr002.c представолен пример использования этих системных вызовов.
Компиляция программе pr002.c представлен пример использования этих системных вызовов.
Компиляция программы pr002.c 
kun@alexandr-Aspire-Z5751:~/OS_2025$ gcc pr002.c -o pr002.out
Запуск на использование программы pr002.out:
kun@alexandr-Aspire-Z5751:~/OS_2025$ ./pr002.out
Идентификатор пользователя: 1185
Индетефикатор пользователя: 1185
