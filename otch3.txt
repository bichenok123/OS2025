Процессы в linux

Идентификация процессов:
Каждый процесс в OC имеет свой индетификатор 
Получить идентификаторр процесса можно при помощи системного вызова getpid()
Получить идентификатор родительского кпроцесса можно при помощи системного вызова getppid()
Прошрамма, выводящая на экран иднтификатолра текущего и родительского пройессов:
#include <stdio.h>
#include <sys/types.h>
#include <unistd.h>
#include <stdlib.h>
int main() {
        pid_t procid, parentprocid;
        procid = getpid() ;
        parentprocid = getppid() ;
        printf("Ид. текущего процесса: %d\nИд. родительского процесса: %d\n", procid, parentprocid);
        return 0;
}
При первом запуске программа видим id текущего проецесса 9074, id родительского процесса - это id командой облочки bash.

При втором запуске программы id текущего процесса поменял сове значение, так как аналогичные процессы завершились, а новый процесс получил новое значение id - 9074
id родительского процесса не изменилось, так как для запуска программы по прежнему исполльзовалась командая оболочка bash.
