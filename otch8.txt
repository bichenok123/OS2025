Проверка записи информации в файл и чтение информации из файла
Для открытия файла ипользуется системный вызов open()
Открытие для записи: fd=open("myfile",O_WRONLY | O_CREAT, -666);
MYFILE - относителльный путь к файлу, O_WRONLY | O_CREAT - флаги для открытия только для записи и создания файла, если он отсутствует, 0666 - права доступа к файлу(чтение/запись для всех пользователей)
Открытие для чтения: fd=open("myfile", O_RDONLY)

Запись информации произволится при помощи системнорго вызова write():
size=write(fd, string, 20);
первый параметр - файловый дискриптор, определяющий средство связи через которое пишется информация
Втрой параметр - источник информции (символьный массив)
третий параметр - количество байт, которое планируется записать
size - значение количества фактичкски записанных в файл байт
Чтение информации производится при помощи системного вызова read();
size = read(fd, string, 20);
fd - файловый дискриптор
string - приёмник информации, куда записывается файловый поток
20 - количество байт, которые требуется прочитать
size - количество фактических прочитанных байт

Системный вызов close() закрывает файд с дискриптом fd
